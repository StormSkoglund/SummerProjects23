Project I: Pong Mod

This is the classic game "Pong". In this game you control the left paddle with the up and down arrow keys. This project consist of code from a tutorial by Devression on Youtube (available at https://www.youtube.com/watch?v=jrqN0uZwhwk&t=307s[accessed on 27nd July, 2023]). Full disclaimer: This is not my work, but I have added this to my projects, in order to learn more about Javascript. I have modded some features in the original code, such as colors, font sizes, as well as the size of the canvas. The canvas has been styled to match the looks of this site.
